# 屏幕截图上传工具 - 集成说明

## 一、快速集成

### 1. 复制核心类
将 `com.screencapture.helper.ScreenCaptureUploadHelper.java` 复制到您的项目中。

### 2. 添加依赖
在您的 app/build.gradle 中添加 OkHttp 依赖：
```gradle
dependencies {
    implementation 'com.squareup.okhttp3:okhttp:4.12.0'
}
```

### 3. 添加权限
在 AndroidManifest.xml 中添加网络权限：
```xml
<uses-permission android:name="android.permission.INTERNET" />
```

## 二、使用方法

### 1. 调用截图上传
在您的 Activity 中调用：
```java
import com.screencapture.helper.ScreenCaptureUploadHelper;

// 调用截图并上传
ScreenCaptureUploadHelper.startCaptureAndUpload(
        activity,
        "http://your-server.com/upload",
        new ScreenCaptureUploadHelper.UploadCallback() {
            @Override
            public void onSuccess(String response) {
                // 上传成功，response 为服务器返回内容
            }

            @Override
            public void onFailed(int errorCode, String errorMsg) {
                // 上传失败
                // errorCode: 1=权限被拒, 2=截图失败, 3=网络错误, 4=服务器错误, 5=正在截图
            }
        }
);
```

### 2. 处理权限结果
在您的 Activity 的 onActivityResult 中添加：
```java
@Override
protected void onActivityResult(int requestCode, int resultCode, Intent data) {
    super.onActivityResult(requestCode, resultCode, data);
    ScreenCaptureUploadHelper.handleActivityResult(requestCode, resultCode, data);
}
```

### 3. 释放资源（可选）
在 Activity 的 onDestroy 中释放：
```java
@Override
protected void onDestroy() {
    super.onDestroy();
    ScreenCaptureUploadHelper.release();
}
```

## 三、完整示例

```java
public class YourActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_your);

        findViewById(R.id.your_button).setOnClickListener(v -> {
            ScreenCaptureUploadHelper.startCaptureAndUpload(
                    this,
                    "http://10.0.2.2:3002/upload",
                    new ScreenCaptureUploadHelper.UploadCallback() {
                        @Override
                        public void onSuccess(String response) {
                            Toast.makeText(YourActivity.this, "成功: " + response, Toast.LENGTH_SHORT).show();
                        }

                        @Override
                        public void onFailed(int errorCode, String errorMsg) {
                            Toast.makeText(YourActivity.this, "失败: " + errorMsg, Toast.LENGTH_SHORT).show();
                        }
                    }
            );
        });
    }

    @Override
    protected void onActivityResult(int requestCode, int resultCode, Intent data) {
        super.onActivityResult(requestCode, resultCode, data);
        ScreenCaptureUploadHelper.handleActivityResult(requestCode, resultCode, data);
    }

    @Override
    protected void onDestroy() {
        super.onDestroy();
        ScreenCaptureUploadHelper.release();
    }
}
```

## 四、注意事项

1. **首次使用**：首次调用会弹出系统权限授权对话框，用户授权后才能截图
2. **服务器地址**：请替换成实际的服务器上传地址
3. **本地存储**：截图不保存本地，仅在内存中转成字节流后上传
4. **线程**：所有回调在主线程执行，可直接更新 UI
