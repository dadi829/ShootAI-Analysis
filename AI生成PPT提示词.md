# AI生成PPT专用提示词

## 使用方法
将以下内容完整复制，粘贴到豆包、ChatGPT、Claude等AI工具中，即可生成PPT内容或代码。

---

## 提示词版本1：生成PPT大纲（适合所有AI工具）

```
请为我生成一份全国大学生计算机设计大赛的演示PPT大纲，具体要求如下：

【项目信息】
- 项目名称：基于端云协同的射击训练智能分析系统
- 团队名称：Dream Chaser
- 团队成员：罗恒、姚静怡
- 指导老师：李艳军
- 演示时长：5分钟 + 3分钟问答
- 总页数：11页

【PPT结构要求】
1. 封面页：项目名称、团队信息、赛事名称
2. 问题背景：行业痛点（技术/市场/合规三维度）
3. 技术方案：端云协同架构图
4. 创新点1+2（合并页）：Prompt工程方案 + 零改造AI升级
5. 创新点3+4（合并页）：端云协同架构 + 多模态数据融合
6. 技术实现：关键算法和流程
7. 性能验证：测试数据和指标
8. 应用价值：场景矩阵和商业化路径
9. 竞品对比：与Scatt等产品的客观对比
10. 团队成果：分工和知识产权
11. 总结展望：技术贡献和未来规划

【设计要求】
- 配色：深蓝色(#1E3A8A) + 科技蓝(#2563EB)
- 字体：思源黑体
- 风格：学术专业、简洁大气
- 创新点页面采用左右分栏布局，每页展示2个创新点

请生成完整的PPT大纲，包含每页的标题、要点内容和设计建议。
```

## 使用建议

### 推荐AI工具
1. **豆包**（字节跳动）：中文理解能力强，适合生成中文PPT内容
2. **ChatGPT**（OpenAI）：逻辑性强，适合生成结构化内容
3. **Claude**（Anthropic）：擅长长文本生成，适合生成完整PPT
4. **文心一言**（百度）：中文优化好，适合国内竞赛场景

### 生成后处理
1. 将AI生成的内容复制到PowerPoint/WPS
2. 应用统一的配色方案和字体
3. 添加图表、流程图等可视化元素
4. 调整布局和动画效果

### 注意事项
- AI生成的内容可能需要微调以符合实际需求
- 建议保留核心数据和技术描述的准确性
- 视觉设计部分可能需要人工优化
