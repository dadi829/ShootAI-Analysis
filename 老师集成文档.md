# 📘 老师集成文档

## 🎯 给老师的集成说明

### 一、老师的App需要做的事情

1. 在界面上加一个按钮，比如叫"AI分析屏幕"
2. 点击按钮时，调用下面的代码唤起我们的App

---

### 二、老师的App调用示例（Kotlin）

```kotlin
import android.content.Intent
import android.net.Uri

// 在按钮点击事件里调用
fun openShootingAnalysis() {
    val intent = Intent(Intent.ACTION_VIEW, Uri.parse("shootinganalysis://upload"))
    intent.addFlags(Intent.FLAG_ACTIVITY_NEW_TASK)
    try {
        startActivity(intent)
    } catch (e: Exception) {
        // 我们的App没安装的情况处理
    }
}
```

---

## 📱 用户使用流程

```
1. 用户在老师的射击训练软件里
2. 用户手动截个图（我们的App暂时需要这一步）
3. 点击"AI分析屏幕"按钮
4. 我们的App自动打开
5. 自动跳到选择截图界面
6. 用户选择刚才截的图
7. 自动上传到网站
8. 刷新网站 http://192.168.31.175:3001 就能看到
```
