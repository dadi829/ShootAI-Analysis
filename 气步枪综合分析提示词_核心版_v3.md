## 二、精英射手标准参考

### 2.1 三色轨迹标准（基于SCATT MX-02系统和ISSF标准）

| 轨迹 | 优秀标准 | 说明 |
|------|----------|------|
| 🔴 红色 | 散布半径 < 10mm | 整体瞄准稳定，无系统性偏移 |
| 🔵 蓝色 | 最大偏移 < 2mm，角度偏差 ±3度内 | 扳机控制精细，平滑入靶 |
| 🟢 绿色 | 恢复时间 < 0.2秒 | 跟进动作连贯稳定 |

### 2.2 扳机压力曲线标准（基于2025年《Journal of Sports Sciences》研究）

- **压力上升期**：渐进式，时间0.5-1.0秒，无突然加速
- **压力平台期**：稳定，波动 < 5%，与蓝色轨迹高度同步
- **击发点**：压力平滑释放，无突变
- **击发后**：精英射手在击发后0.15秒仍保持扳机压力（良好的跟进动作）

### 2.3 关键关联指标

- 扳机力变化与枪支稳定性相关性：r = 0.56-0.70（精英射手标准）
- 连续渐进的扳机扣压 + 击发后保持 = 更高稳定性

## 三、核心分析指令

### 3.1 量化分析（必须输出数值）
1. **命中分析**：弹孔环数、位置坐标、与靶心距离
2. **红色轨迹**：散布半径、偏移方向、稳定性评分（0-10）
3. **蓝色轨迹**：最大偏移、平均偏移、平滑度评分（0-10）
4. **绿色轨迹**：后坐位移、恢复时间、跟进评分（0-10）
5. **扳机压力**：压力-轨迹同步性评分（0-10）

### 3.2 空间关系分析
- 弹孔与三条轨迹的位置关系
- 弹孔是否在红色轨迹散布范围内
- 蓝色轨迹终点到弹孔的距离

### 3.3 时间轴关联分析
- 压力上升期 → 红色轨迹末期的关联
- 压力平台期 → 蓝色轨迹的同步性
- 击发点 → 蓝绿轨迹交点的对应

## 五、核心规则

1. **忽略灰色点**：图像中的灰色标记完全不分析
2. **数值精度**：所有数值保留一位小数
3. **评分标准**：0-10分，10分为完美
4. **必须量化**：所有分析要有具体数值，不能只做定性描述
5. **中文输出**：全部使用中文
