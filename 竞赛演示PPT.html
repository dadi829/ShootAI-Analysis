🧮 核心算法

欧几里得距离环数判定：

distance = √(x² + y²)
if distance ≤ 3.75mm → 10.9环
if distance ≤ 7.5mm → 10.5环
...

🛡️ 容错机制

  • 多级重试策略：1s → 3s → 5s指数退避
  • JSON解析容错：代码块提取+逐步截断
  • 超时控制：90秒读超时，AbortController中断
  • 连接复用：HTTPS Agent，maxSockets=10
